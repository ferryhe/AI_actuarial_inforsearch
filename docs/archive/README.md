# Archived Documentation

This directory keeps historical project material for traceability. Archived documents are **not** the current implementation checklist unless a current guide links to them for background.

## Subdirectories

- `plans/`: completed or superseded planning documents.
- `security/`: legacy security review notes and old quick-fix guides.
- `guides/`: legacy quick-start, modular-system, and chatbot roadmap guides that predate the current FastAPI + React product contract.
