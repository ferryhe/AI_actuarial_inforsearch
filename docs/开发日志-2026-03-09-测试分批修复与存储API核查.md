## 技术决策

- 旧 schema 不直接要求手工迁移，优先在运行时补齐缺列，降低历史数据库升级门槛。
- 测试中的 `.env` 与 YAML 路径必须彻底隔离，否则会让本地开发环境污染 CI/本地回归结果。
- 聊天链路里凡是会被测试 patch 的对象，都不应在 `create_app()` 阶段固化为闭包引用。
- `REQUIRE_AUTH=false` 不是“后台全公开”，仅意味着允许匿名浏览和公共聊天；管理类 API 仍需认证。
