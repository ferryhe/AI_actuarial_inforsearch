# 开发日志 - 2026-03-09 - FastAPI 入口与兼容网关

## 进度摘要

本阶段在 `feature/react-fastapi-migration` 分支完成了 FastAPI 第一批落地：新增 FastAPI 网关入口，用它作为 React 未来的后端入口，并通过兼容挂载方式回退到现有 Flask 应用，避免迁移初期立即切断现有功能。

- 更新 `README.md`，补充 FastAPI 网关启动方式

## 技术决策

- 使用 `a2wsgi.WSGIMiddleware` 挂载 legacy Flask，避免继续依赖 Starlette 已弃用的 WSGI 中间件。
- FastAPI 当前只承接“网关职责”和少量原生路由；未迁移接口仍回落到 Flask。
- 先保证入口能运行和可测试，再逐批把 `/api/*` 路由从 Flask 迁到 FastAPI。
