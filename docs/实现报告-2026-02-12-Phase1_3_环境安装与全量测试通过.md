## 🔍 Troubleshooting Guide

- 若报错 `No module named tiktoken`：
  - 执行 `.\.venv\Scripts\python.exe -m pip install tiktoken`
- 若 pytest 报 `--cov` 参数无法识别：
  - 执行 `.\.venv\Scripts\python.exe -m pip install pytest-cov`
- 若 Windows 下清理临时目录报 `PermissionError`：
  - 通常是 SQLite 句柄未即时释放，等待后重试即可
